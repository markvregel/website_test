GUI Insructions

Import spectral images and LiDAR data

The classification of the vegetation will be based on varying spectral reflectance. Therfore it is neccassry to upload high resolution spectral images. The favoured data to use are the spectral imagery of rijkwaterstraat and Geodesk. If these are not available change data format. If available for the year you are looking for you can also integrate elevation point cloud data for getting better classification results, when pressing (putpic). The program is made for integration of the ActueelHoghtebestand Netherlands LiDARpointcloud data. The data is processed for the pixelsize used by the spectral imagery and adds more bands to the images learn more about adding LiDAR pictures Otherwise import point cloud data.

Here is more detailed information on the spectral imagery and on the point cloud data

Classification

To classify the vegetation the calculation algorythm has to be trained. Therefore, you can either load up classification shape files from ArcGis or you can add the polygons for the classes directly in R. When classifying it is important to set small polygons and not too many otherwise the amount of data is too big to calculate. What you need: classification shapefiles made with ArcGis and Spectral images (maybe with Included LiDAR Data

Comparison of vegetation cover between different years

The first resulting map shows where changes occure in the area. The changing areas are red by default and the stable ones are green. Due to different spectral reflection of the sunlight in different times of the day and years the years will always differ.

On the right side on top there is a box where all options are ticked. Untick the lowest box. The map changes and gets many classes. Here the change of every class is shown. For more intuitive understanding go to the Vegetation Change classes map. The changes of one class are represented. The classes which change to the considered class have various blue colours. When the considered class changes into another class it is shown in shades of orange.

The comparison is done by substraction of two maps.
